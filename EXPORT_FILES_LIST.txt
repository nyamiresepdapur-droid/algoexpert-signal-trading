METODE 1: Manual Copy-Paste
1. Buka setiap file di editor
2. Select All (Ctrl+A)
3. Copy (Ctrl+C)
4. Buka Bolt.new
5. Buat file baru dengan nama yang sama
6. Paste (Ctrl+V)

1. Push project ke GitHub
2. Di Bolt.new, import from GitHub
